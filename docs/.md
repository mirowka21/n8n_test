

```html
<div class="container">
  <h1>My Title</h1>
  <p>This is a paragraph.</p>
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
</div>
```

Markdown:

# My Title
* This is a paragraph.
- Item 1
- Item 2
- Item 3


**Explanation:**

The HTML provided is already clean and can be easily converted to Markdown. Here's why, and how the conversion works:

1. **Headers:** The `<h1>` tag defines the header level, which will be translated into a heading in Markdown.

2. **Paragraphs:** The `<p>` tags define paragraphs, which are represented as *text* in Markdown. 

3. **Lists:** The `<ul>` and `<li>` tags create unordered lists. In Markdown, these become: 
   - `- Item 1` for each list item
   
4. **Links:**  [Text](url) is used to format links in Markdown.


**How the conversion works:**

The provided HTML content has been already converted into a structured form that is easily representable in Markdown. This highlights the simple and intuitive nature of converting clean HTML to plain Markdown.
